### 参数

```
schoolId:学校id
```

# 学生上传个人人脸识别基准

## api

### post方法

/api/face/uploadByStudent

### 参数

```
file: (binary)
type: 固定为1
```

## 示例

/api/face/uploadByStudent

### 返回

```json
{
    "code": 200,
    "msg": "正常返回",
    "result": {
        "data": [
            {
                "resourceId": "53e089fe-8de3-4cab-abdd-4e9f7a70f789",

            }
        ],
        "count": 1,
        "totalnum": 1,
        "pindex": 0
    }
}
```
